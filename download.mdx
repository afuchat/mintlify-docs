## System Requirements

### Android

| Requirement | Minimum |
|---|---|
| Android version | Android 8.0 (Oreo) or later |
| Storage | ~80 MB free space for installation |
| Internet | Wi-Fi or mobile data (3G or faster recommended) |
| Google Play | Google Play Services required |

For the best experience — especially for real-time chat, voice messages, and video content — a stable 4G or Wi-Fi connection is recommended.

### Web Browser

AfuChat's web app runs entirely in your browser. No plugins or extensions are needed.

| Browser | Supported versions |
|---|---|
| Google Chrome | Version 90 or later |
| Mozilla Firefox | Version 88 or later |
| Microsoft Edge | Version 90 or later |
| Safari | Version 14 or later (macOS and iOS) |
| Opera | Version 76 or later |

Enable JavaScript and allow browser notifications if you'd like to receive real-time message alerts while the tab is open.

<Note>
  AfuChat is completely free to download and free to create an account. Premium subscriptions (Silver, Gold, Platinum) and ACoin top-ups are optional in-app purchases available after you sign up.
</Note>
